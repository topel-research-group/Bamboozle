# Pipeline_vilma

This is a pipeline that will take raw data, as fastq files, or BAM files as input and give a filtered output.

The steps will include mapping, SNP calling, Fst statistics, filtering and plotting.

### Dependencies
Versions used:  
Bowtie2/v2.3.3.1  
samtools/v1.9  
bcftools/v1.9  
snpEff/v.4.3t
